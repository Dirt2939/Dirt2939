# Sobre Mim

Sou estudante de Engenharia de Software e desenvolvedor em formação, com foco em desenvolvimento full stack, Java, APIs REST e construção de interfaces web. Tenho interesse em criar sistemas bem organizados, com regras de negócio claras, boa experiência de uso e estrutura preparada para evolução.

Meus projetos reúnem estudos acadêmicos e aplicações práticas envolvendo Java, Programação Orientada a Objetos, Java Swing, MySQL, HTML, CSS, JavaScript, TypeScript, Express, Prisma, Firebase Authentication, Canvas, Web Audio API, Phaser e Socket.IO.

Atualmente, meu foco está em:

- Desenvolvimento de aplicações web full stack com frontend, backend e banco de dados.
- Criação de APIs REST com TypeScript, validação de dados e boas práticas de organização.
- Projetos em Java com Programação Orientada a Objetos, Swing, MVC, DAO e MySQL.
- Interfaces interativas com JavaScript, manipulação de dados e experiências no navegador.
- Estudos sobre integração de IA em produtos digitais de forma prática e útil.

### Aberto a

- Estágios em desenvolvimento de software.
- Oportunidades em desenvolvimento full stack.
- Projetos com foco em backend, APIs e banco de dados.
- Colaborações em projetos web, Java e aplicações educacionais.
